## ⌨️ 🔀 🔊 어떤 서비스인가요?
_____
- `ChatSound` 는 채팅과 같은 텍스트 데이터를 입력받아, 텍스트에 맞는 소리를 생성해 주는 서비스입니다!

- `ChatSound` 를 이용하면 `🐕개 짖는 소리`, `🚪문 여는 소리`, `👶아기 우는 소리` 등 다양한 소리를 생성할 수 있어요. 그것을 이용해, 국내에 몇 없는 폴리아티스트, 전문 소리 제작업체 등을 도와줄 수도 있습니다!

## 🏃‍실행하기

```zsh
mvn clean
mvn package
mvn run
```
